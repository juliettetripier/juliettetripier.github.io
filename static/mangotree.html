CIB Mango Tree

Summary

CIB Mango Tree is a project sponsored by my volunteer group, Civic Tech DC.

CIB Mango Tree is a collaborative, open-source software tool to detect coordinated inauthentic behavior (CIB) in online datasets.

The CIB Mango Tree project is split into two main parts: the website, which serves primarily as an educational resource, and the CLI tool itself, which contains the battery of tests designed to measure CIB activity.

I serve as a member of the web development team and am mainly responsible for translating our Figma layouts into code. I also work alongside the CLI team to develop offline HTML output for our analysis tools.
