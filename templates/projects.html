Eggsplorer

Github Repo README

Eggsplorer is a full-stack, Python-based Flask application that allows users to find new restaurants and track their restaurant visits.

Features include:

  • User creation and login
  • Restaurant search via Yelp Fusion API
  • User-created wishlists and system for tracking restaurant visits
  • Tagging system for restaurants
  • User achievements and achievement leaderboard
  • Unlockable cosmetics for site mascot
Technologies

Front End: HTML, CSS, Jinja2, JavaScript (AJAX, JSON), Bootstrap, Flexbox

Back End: Python, Flask, Yelp Fusion API

Database: PostgreSQL, SQLAlchemy

Multiplayer Tron

Site Github Repo

Multiplayer Tron is a Tron clone built from scratch using pure JavaScript and socket-based networking for remote multiplayer capabilities. This project is still in progress.

Completed features

  • Collision detection
  • Win vs. loss detection
  • Single-player game flow versus an AI opponent

In progress

  • Multiplayer lobby
  • Multiplayer game flow
Technologies

Front End: HTML, CSS, JavaScript

Back End: Node.js, Fastify, Websocket
